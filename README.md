# Yandex Disk API Automation Tests

Репозиторий содержит автоматизированные тесты для проверки базовой логики REST API Яндекс Диска. Архитектура построена с использованием паттерна API Object для обеспечения масштабируемости и читаемости.

# Что проверяет тест
Реализован комплексный сценарий (End-to-End), который за один прогон проверяет полный жизненный цикл ресурса:
1. **Создание ресурса** (PUT) — создание папки с уникальным ID.
2. **Верификацию** (GET) — проверку метаданных созданного объекта.
3. **Загрузку контента** (POST) — асинхронную загрузку файла по внешнему URL в созданную папку.
4. **Очистку данных** (DELETE) — удаление всей тестовой структуры.
5. **Негативный кейс** (GET 404) — подтверждение отсутствия данных после удаления.

# Как запустить (Quick Start)

### 1. Подготовка окружения и установка
Убедитесь, что у вас установлен Python 3.10+.

```bash

git clone https://github.com/0su0000000i17/yandex_disk_test_api.git
cd yandex_disk_test_api

# Создать и активировать виртуальное окружение
py -m venv venv
source venv/Scripts/activate  # Для Windows
# source venv/bin/activate    # Для macOS/Linux

# Установить зависимости
pip install -r requirements.txt

# Создать файл .env

# Запуск тестов
pytest -v -s
